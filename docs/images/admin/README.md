# Captura de Screenshots do Treinamento Admin

Use esta estrutura para salvar as imagens que entram no guia `treinamento-admin-primeira-semana.md`.

## Padrao recomendado

- Salve em PNG
- Tente manter a mesma largura visual entre telas
- Evite capturar dados sensiveis reais
- Se possivel, use uma base de testes
